CÓMO USAR:
═══════════════════════════════════════════════════════════════════════════════

1. INSTALAR:
   pip install PyPDF2 PyMuPDF Pillow

2. EJECUTAR GUI (Recomendado):
   python gui_main.py

3. CLI - EXTRACCIÓN:
   python cli_extract.py entrada.pdf salida.pdf "1-3,5"

4. CLI - CONVERSIÓN:
   python cli_convert.py ./pdfs --zoom 2.0 --quality 95

5. DESDE PYTHON:
   from src.services import PDFExtractor, PDFToImageConverter
   result = PDFExtractor.extract_from_string(...)
